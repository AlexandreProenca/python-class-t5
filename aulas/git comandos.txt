Comandos basicos do Git

Importante! os comandos devem ser digitados no diretório GIT

git clone          # baixa o repositorio da origem
git status
git log

# entra no diretório 
	git add    #adiciona o arquivo no stage
	git commit # salva um ponto no tempo
	git push   # envia as modificações
	git pull   # pega as atualizaçãoes
